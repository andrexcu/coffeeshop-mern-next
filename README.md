### Prerequisites

**Node version 14.x**

### Cloning the repository

```shell
git clone https://github.com/andrexcu/store-with-next.git
```

### Install packages

```shell
npm i
```

### Client: Setup .env file


```js
NEXT_PUBLIC_API_URL=
```

### Server: Setup .env file


```js
DATABASE_URL=
NODE_ENV=
REFRESH_TOKEN_SECRET=
ACCESS_TOKEN_SECRET=
```

### Start the app

```shell
npm run dev
```
